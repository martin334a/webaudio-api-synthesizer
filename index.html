Web Audio API Matrix Synthesizer

This web app uses the Web Audio API to change the frequency and pitch of a single mp3 file in order to create a dynamic synthesizer. Up to the release of iOS6 this kind of implementation would not have been possible, and would have required different audio files, which would have made any musical (tempo-based) representation fairly difficult due to buffering and preloading of audio. A superimposed canvas allows for multi-touch interactions on the interface.
